# Topic: Test Automation Basics (Assertions)

## Key Concepts
* **Tests Tab:** JavaScript code that runs *after* the server responds.
* **pm.* Library:** Postman's object for writing tests.

## Assertion 1: Status Code Check
```javascript
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

```
## Assertion 2: Response Body Check
const response = pm.response.json(); // Parse JSON body
pm.test("Status property value is OK", function () {
    pm.expect(response.status).to.equal("OK");
});
// Gotcha: Use bracket notation for hyphenated properties: response['current-stock']
