# Topic: JSON Structure and Dynamic Data Generation

## JSON Structure
* **Rule:** Keys **must** be double-quoted strings.
* **Rule:** Values (strings) **must** be double-quoted. Numbers and Booleans (`true`/`false`) are not quoted.

## Dynamic Variables (Random Data)
* **Purpose:** Generate unique test data to avoid data collision (e.g., registering same client twice).
* **Syntax:** Use built-in dynamic variables with the `$` prefix.
* **Example:**
    ```json
    {
      "clientName": "{{$randomFullName}}",
      "clientEmail": "{{$randomEmail}}"
    }
    ```
