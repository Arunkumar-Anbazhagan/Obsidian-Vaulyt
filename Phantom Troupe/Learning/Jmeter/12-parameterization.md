# 12. Parameterization    
## Section 12: Parameterization (Data-Driven Testing)   
### Using CSV Data Set Config   
### CSV File Format   
```
username,password
user1,pass1
user2,pass2
user3,pass3

```
### Configuration   
- Variable Names: username,password   
- Use in requests: ${username}, ${password}   
   
### Advanced Parameterization   
- Multiple CSV files   
- Different sharing modes   
- Recycle vs Stop options   
- Random order from CSV
