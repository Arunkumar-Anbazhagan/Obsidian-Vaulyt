# 8. Assertion   
## Section 8: Assertions   
### Purpose   
- Validate response data   
- Mark requests as pass/fail based on conditions   
   
### Types of Assertions   
### Response Assertion   
- Check response text, code, message, headers   
- **Patterns**:   
    - Contains   
    - Matches (regex)   
    - Equals   
    - Substring   
- **Apply to**: Main sample, Sub-samples, Both   
   
### Duration Assertion   
- Assert response time within threshold   
- Example: Response time < 2000ms   
   
### Size Assertion   
- Assert response size (bytes)   
- Equals, Greater than, Less than   
   
### JSON Assertion   
- Assert JSON path expressions   
- Validate JSON response structure/values   
   
### XPath Assertion   
- Assert XML using XPath   
- Validate XML response   
   
### HTML Assertion   
- Parse HTML and assert format   
- Check errors, warnings   
   
### XML Schema Assertion   
- Validate XML against XSD schema   
   
### Compare Assertion   
- Compare with previous results
