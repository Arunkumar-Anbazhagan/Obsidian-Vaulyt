# 21. Plugins   
## Section 21: Plugins   
### JMeter Plugins Manager   
1. Download `jmeter-plugins-manager.jar`   
2. Place in `lib/ext` folder   
3. Restart JMeter   
4. Options > Plugins Manager   
   
### Popular Plugins   
- **Custom Thread Groups**: Ultimate Thread Group, Stepping Thread Group   
- **3 Basic Graphs**: Response Times, Transactions per Second, Active Threads   
- **Custom JMeter Functions**: More functions like random date   
- **Parallel Controller**: Execute requests in parallel   
- **Throughput Shaping Timer**: Control load pattern   
- **Dummy Sampler**: For debugging   
- **WebSocket Samplers**: WebSocket testing
