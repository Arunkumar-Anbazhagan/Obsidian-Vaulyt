# 3. Jmeter GUI   
## Section 3: JMeter GUI Overview   
### Main Components   
- **Menu Bar**: File, Edit, Run, Options, Help   
- **Toolbar**: Quick access buttons (New, Open, Save, Run, Stop)   
- **Test Plan Tree**: Left panel showing test plan hierarchy   
- **Right Panel**: Configuration details of selected element   
- **Status Bar**: Shows test execution status   
   
### Important Settings   
- **Options > Look and Feel**: Change UI theme   
- **Options > Choose Language**: Change interface language   
- **Test Plan > Run Thread Groups Consecutively**: Sequential execution   
- **Test Plan > Run Teardown Thread Groups After Shutdown**: Cleanup execution
