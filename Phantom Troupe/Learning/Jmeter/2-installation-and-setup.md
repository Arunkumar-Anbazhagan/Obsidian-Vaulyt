# 2. Installation and setup   
## Section 2: JMeter Installation & Setup   
### Prerequisites   
- Java JDK 8 or higher (JDK recommended over JRE)   
- Verify Java: `java -version`   
   
### Installation Steps   
1. Download JMeter from Apache website (latest binary version)   
2. Extract the zip/tar.gz file to desired location   
3. Navigate to `/bin` folder   
4. Run `jmeter.bat` (Windows) or `jmeter.sh` (Linux/Mac)   
   
### JMeter Folder Structure   
- **bin/**: Executable files, configuration files   
- **lib/**: JMeter jar files   
- **lib/ext/**: External libraries and plugins   
- **extras/**: Additional tools (Ant, Jenkins integration)
