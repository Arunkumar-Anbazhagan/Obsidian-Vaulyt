#  5. HTTP Request Sampler   
## Section 5: HTTP Request Sampler (Detailed)   
### Configuration Fields   
- **Protocol**: http or https   
- **Server Name/IP**: Target server   
- **Port Number**: Default 80 (http), 443 (https)   
- **Method**: GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS   
- **Path**: URL path   
- **Parameters**: Query parameters or POST data   
- **Body Data**: Raw request body (JSON, XML)   
- **Files Upload**: Upload files with request   
   
### Advanced Settings   
- **Redirect Automatically**: Follow redirects   
- **Follow Redirects**: Follow redirects manually   
- **Use KeepAlive**: Reuse connections   
- **Use multipart/form-data**: File uploads   
- **Browser-compatible headers**: Add appropriate headers
